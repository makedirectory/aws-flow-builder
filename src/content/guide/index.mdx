# User Guide

How to use AWS Flow Builder to design and understand cloud architecture.

- **Build a diagram** — drag services from the palette onto the canvas and
  connect them with typed relationships.
- **Import infrastructure** — load a CloudFormation template or Terraform
  `show -json` output and see it rendered automatically.
- **Validate** — run architecture checks and best-practice rule suggestions.
- **Save & load** — persist diagrams to the server and reopen them later.

More pages are added here as features land.
